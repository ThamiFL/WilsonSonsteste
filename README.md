# 🚢 Sistema de Agendamento de Visitas – Wilson Sons

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de facilitar a organização e o agendamento de visitas para eventos promovidos pela Wilson Sons, uma das maiores operadoras de logística portuária e marítima do Brasil. A plataforma permite que participantes realizem suas inscrições de forma simples, rápida e intuitiva, proporcionando uma melhor gestão do evento e das vagas disponíveis. A Wilson Sons atua há mais de 180 anos no setor de logística portuária e marítima, oferecendo soluções integradas para o comércio nacional e internacional. :contentReference[oaicite:0]{index=0}

## 🎯 Objetivo

Desenvolver uma solução moderna e responsiva para:

- Realizar inscrições em eventos;
- Organizar visitantes e participantes;
- Centralizar informações sobre o evento;
- Melhorar a experiência dos usuários durante o processo de agendamento;
- Facilitar o gerenciamento dos participantes.
